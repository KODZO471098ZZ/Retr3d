Documentation

Purpose

Slic3r converts STL files into printable files. Retr3d uses Slic3r to make it that much easier to start printing as soon as possible

Installation

First, download the precompiled Slic3r packages here. Next, extract the files into the "slic3r" file included in Retr3d. Huzzah! You have successfully installed Slic3r!

Configuration

To start customizing your printing settings, hit up the great manual written by Slic3r's developers.

Slic3r Integration with Repetier

Slic3r is included in Repetier. Repetier stores all of its Slic3r configuration files in HOME/AppData/Roaming/Slic3r resp. $HOME/.slic3r for Linux users. Be aware that AppData is set as invisible folder and you need to enable viewing ob invisible files in the file explorer to see it.
